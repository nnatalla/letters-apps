ℹ️ Instrukcja obsługi:

  • Zbieg komorniczy: Gdy wybierzesz opcję "ZBIEG KOMORNICZY", system wygeneruje osobny list dla każdego komornika
  • Nawigacja: Używaj przycisków "Poprzedni/Następny" lub klawiszy strzałek ← → do przeglądania listów
  • Pobieranie: Możesz pobrać listy w formacie DOC lub PDF - pojedynczo lub wszystkie na raz
  • Formaty plików:
    • 📄 DOC - plik Microsoft Word, można edytować w programach tekstowych
    • 📃 PDF - plik w formacie PDF gotowy do wydruku
  • Kopiowanie: Kliknij "📋 Kopiuj treść" aby skopiować tekst listu do schowka
  • Edycja daty: Użyj "📅 Zmień datę" aby dostosować datę w liście
